【裁判字號】    95,易,700
【裁判日期】    951019
【裁判案由】    恐嚇取財
【裁判全文】
臺灣彰化地方法院刑事判決               95年度易字第700號
公  訴  人  臺灣彰化地方法院檢察署檢察官
被      告  甲○○
上列被告因恐嚇取財案件，經檢察官提起公訴（95年度偵字第44
50號），本院判決如下：
    主    文
甲○○連續意圖為自己不法之所有，以詐術使人將本人之物交付
，累犯，處拘役拾日，如易科罰金，以佰元折算壹日；又連
續以脅迫使人行無義務之事，累犯，處拘役貳拾日，如易科罰金
，以佰元折算壹日；又連續意圖為自己不法之所有，以恐嚇使
人將本人之物交付，累犯，處有期徒刑捌月。拘役部分，應執行
拘役肆拾日，如易科罰金，以佰元折算壹日。
    犯罪事實
一、甲○○前曾於民國92年間，因搶奪案件，經臺灣高雄地方法
    院以92年度訴字第576 號，判處有期徒刑1 年6 月確定，入
    監執行後，於93年8 月15日縮短刑期執行完畢。詎其仍不知
    悔改，竟因見乙○○所經營之鹹酥雞攤位有現款出入，且為
    貪圖免費獲得鹹酥雞食用，竟基於意圖為自己不法所有之概
    括犯意，於附表貳所示之時間、地點，以附表貳所示之詐騙
    方式，連續5 次對乙○○詐騙如附表貳所示金額之鹹酥雞，
    致乙○○陷於錯誤，誤以為甲○○確有支付現款之意願，而
    將烹調完成之鹹酥雞交予甲○○；甲○○另行起意，基於使
    人行無義務事之概括犯意，明知乙○○並無借款予其之義務
    ，竟因欠款花用，於附表所示之時間、地點，均以附表
    所示之脅迫方式，使本無借款義務之乙○○因恐自身安全及
    攤位無法順利經營，而為其本無義務之借款行為3 次，甲○
    ○則於事後歸還借款；甲○○另行起意，基於意圖為自己不
    法所有之概括犯意，於附表肆所示之時間、地點，以附表肆
    所示之恐嚇方式，向乙○○為惡害之通知，致乙○○因甚感
    害怕，決定交付如附表肆所示之財物。嗣因甲○○於附表肆
    編號4 所示之時間，向乙○○恐嚇取得財物後，仍於乙○○
    之攤位附近徘徊，乙○○因恐甲○○傷人始報警，經警在現
    場將甲○○逮捕而查知上情。
二、案經乙○○訴由彰化縣警察局鹿港分局報告臺灣彰化地方法
    院檢察署檢察官偵查起訴。
    理    由
壹、證據能力部分：
    本案全部卷證所涵括之供述證據及非供述證據，公訴人及被
    告甲○○均同意全部證據之證據能力，是後述所引用證據之
    證據能力均無疑義，合先敘明。
貳、認定事實及論罪科刑部分：
一、訊據被告甲○○固坦承曾至證人乙○○所經營之鹹酥雞攤位
    購買鹹酥雞，且向證人乙○○拿取款項，並曾對證人乙○○
    說是竹聯幫份子，如果不給錢，就要讓其關店等語，惟矢口
    否認涉有前開犯行，辯稱：伊未曾加入幫派，伊說這些話只
    是開玩笑云云。惟查：上揭被告詐欺取得鹹酥雞、使證人乙
    ○○行無義務之借款行為，及意圖為自己不法所有，向證人
    乙○○索取現金花用等情，業據證人即被害人乙○○於警詢
    及本院審理時證稱：被告係從94年12月中旬開始向其購買鹹
    酥雞，本來有付錢，後來到95年2 月開始至95年3 月止，被
    告就先跟其說要買鹹酥雞，等其炸完將鹹酥雞交給被告後，
    被告就會說沒錢，且說是混竹聯幫的，不付錢要賒帳，就將
    鹹酥雞拿走，事後也沒付款，次數總共5 次，金額為新臺幣
    （下同）60元至160 元不等；被告曾經在95年5 月間，向其
    要錢，如其面有難色，被告就說交錢，否則要其難活下去，
    而其中5 次，被告借款後未歸還，金額為200 元至300 元不
    等，被告又在95年5 月8 日晚上，曾向其拿200 元，95年 5
    月9 日，先向其要了20元，隔了40分鐘，又向其要了10元，
    被告每次向其要錢，臉都很兇，並說不給錢要讓其關店，這
    些款項也沒歸還；被告另外曾經向其借錢有還的有3 次，金
    額分別是160 元、200 元及150 元；被告每次向其借錢，其
    都是因為被告說話時臉看起來很兇，且說曾經說過混竹聯幫
    ，害怕遭到報復且對其不利，才會出借等語（見警卷第 3頁
    及本院審判筆錄），衡之證人乙○○自警詢至本院審理時，
    就被告所為前開行為，均指證歷歷，且被告及證人乙○○均
    陳稱雙方素無怨隙，而證人乙○○於本院審理時復證稱：其
    也不是很想告被告，會去報警，實在是因為被告常常在攤位
    附近停留且大小聲，怕被告傷人，很恐怖等語（見本院審判
    筆錄），因可認證人乙○○實無故意誣陷被告之可能及必要
    ，況證人乙○○於本院審理時，於具結負偽證罪處罰之心理
    壓力下，仍為前開證述，而被告就證人乙○○所述之客觀情
    狀亦不否認，實可認證人乙○○前開證述，係屬真實，此外
    ，尚有證人乙○○所經營之鹹酥雞攤位照片，及證人乙○○
    親自於95年5 月9 日晚上11時30分前往報案之彰化縣警察局
    鹿港分局受理刑事案件報案三聯單各1 紙在卷可稽（見警卷
    第7 頁），前情已堪認定。至被告雖辯稱伊說這些話是開玩
    笑的云云，惟查，證人乙○○於偵訊及本院審理時均證稱：
    被告的臉看起來很兇，不像是開玩笑的等語，且被告現年23
    歲，此有年籍資料可稽，又身強體壯，伊為前揭陳述，實難
    令人相信僅係開玩笑之言；況據被告及證人乙○○證述，其
    等2 人僅係一般攤販及顧客關係，倘被告所稱「混竹聯幫」
    、「不借錢要讓其關店活不下去」等語純係開玩笑，則經營
    小本生意之證人乙○○焉有任由被告拒不付款，且借款予被
    告，及交付現款供被告花用之可能。綜上所述，被告前開辯
    解，應係事後卸責之詞，不足採信。本案被告連續詐欺取財
    、連續以脅迫方式使人行無義務之事，及連續恐嚇取財犯行
    ，事證均已明確，被告犯行洵堪認定，應予依法論科。
三、我國刑法於94年2月2日修正公布，95年7月1日施行（下稱新
    法；修正前刑法下稱舊法）後，依最高法院95年度第 8次刑
    事庭會議決議決議內容，有關新舊法之適用原則如下：
    (一)新法第2條第1項之規定，係規範行為後法律變更所生新舊
      法律比較適用之準據法，於新法施行後，應適用新法第 2
      條第1項之規定，為「從舊從輕」之比較。
    (二)基於罪刑法定原則及法律不溯及既往原則，行為之處罰，
      以行為時之法律有明文規定者為限，必行為時與行為後之
      法律均有處罰之規定，始有新法第2條第1項之適用。
    (三)比較時應就罪刑有關之共犯、未遂犯、想像競合犯、牽連
      犯、連續犯、結合犯，以及累犯加重、自首減輕暨其他法
      定加減原因（如身分加減）與加減例等一切情形，綜其全
      部罪刑之結果而為比較。
    (四)新法第33條第5 款規定罰金刑為新臺幣1,000 元以上，以
      百元計算之，新法施行後，應依新法第2 條第1 項之規定
      ，適用最有利於行為人之法律。
四、查被告先後5 次，先向證人乙○○佯稱欲購買鹹酥雞，待證
    人乙○○誤以為被告有支付款項之意願，而陷於錯誤，並將
    鹹酥雞交予被告後，被告始出言混竹聯幫等語，拒絕付款賒
    帳，被告此部分所為，係犯刑法第339 條第1 項之詐欺取財
    罪。公訴意旨認被告此部分應係犯刑法第346 條第1 項之恐
    嚇取財罪，惟證人乙○○此5 次交付鹹酥雞，均係因誤以為
    被告有付款之意願，而被告均係在取得鹹酥雞後，始出言不
    遜等情，業據證人乙○○於本院審理證述綦詳，是證人乙○
    ○尚非因被告之恐嚇行為，而交付財物，而係因誤以為被告
    有支付現款之意願，而交付鹹酥雞，是公訴意旨認此部分應
    依刑法第346 條第1 項恐嚇取財罪論處，尚有未洽，惟此部
    分之其中1 次犯行，於起訴書業已載明，且起訴之社會基本
    事實相同，爰就此部分依法變更起訴法條；又按刑法第 304
    條第1 項之強制罪，與刑法第346 條第1 項之恐嚇取財罪，
    其構成要件迥然不同，前者無不法所有之意圖，後者則以意
    圖自己或第三人之不法所有為前提條件。查被告先後3 次，
    於附表所示之時、地，以脅迫之方式，使本無借款意願之
    證人乙○○，因心生畏佈，而分別借款160 元、200 元及15
    0 元予伊，被告既已於事後將此部分借款清償，應足認被告
    並無不法所有之意圖，核被告此部分所為，應係犯刑法第30
    4 條第1 項之強制罪。公訴意旨認被告此部分應係犯刑法第
    346 條第1 項之恐嚇取財罪，惟被告既無不法所有之意圖，
    即與恐嚇取財罪之構成要件相間，惟此部分之其中2 次犯行
    （即附表編號2 、3 部分），於起訴書業已載明，且起訴
    之基本事實相同，爰就此部分依法變更起訴法條。又被告意
    圖為自己不法所有，以附表肆所示之脅迫話語，致證人乙○
    ○心生畏懼，而分別交付現款共8 次，核被告此部分所為，
    則係犯刑法第346 條第1 項之恐嚇取財罪。又被告行為後，
    刑法關於連續犯、累犯、易科罰金、主刑罰金刑及定執行刑
    等規定均有修正，依前揭最高法院95年度第8 次刑事庭會議
    決議內容，應綜其全部罪刑之結果而為比較後，始得判斷應
    適用新法或舊法，經本院為如附表壹所示之比較後，認本件
    依刑法第2 條第1 項前段，就以下連續犯、累犯、易科罰金
    、主刑罰金刑及定應執行刑等適用，應依舊法。被告先後5
    次詐欺取財、3 次以脅迫使人行無義務事，及8 次恐嚇取財
    犯行，均時間緊接，分別觸犯構成要件相同之罪，顯各係基
    於概括犯意為之，依修正刪除前刑法第56條之規定，為連續
    犯，應各以1 罪論，並均分別依法加重其刑。被告所犯前開
    連續詐欺取財、連續以脅迫使人行無義務事，及連續恐嚇取
    財3 罪間，犯意各別，罪名不同，應予分論併罰。又公訴人
    起訴事實雖僅敘及被告曾1 次佯稱購買鹹酥雞後拒絕付款，
    而未敘及被告其餘4 次詐欺取財犯行；且僅敘及被告曾2 次
    強制證人乙○○借款（即如附表編號2、3），而未敘及被
    告亦曾於附表編號1所示之時、地強制借款之犯行，然前
    開部分事實，與公訴人起訴之詐欺及強制犯罪事實，既分別
    具有連續犯之裁判上一罪關係，為起訴效力所及，本院自應
    併予審理。再被告前曾於92年間，因搶奪案件，經臺灣高雄
    地方法院以92年度訴字第576號，判處有期徒刑1年6月確定
    ，入監執行後，於93年8月15日縮短刑期執行完畢等情，此
    有臺灣高等法院被告前案紀錄表存卷可參，其於5年內再犯
    本件最重本刑為有期徒刑以上之前開3罪，皆為累犯，應分
    別依修正前刑法第47條之規定，遞加重其刑。爰審酌被告正
    值壯年，竟不思循正當途徑或取財物，竟利用其身型優勢，
    屢對經營小本生意之證人乙○○詐欺取財、強制借款及恐嚇
    取財，惟其尚僅以口頭脅迫，且所獲得財物非鉅，犯後亦已
    表明知錯等一切情狀，核情分別量處如主文所示之刑，且就
    拘役刑部分，依修正前刑法第41條第1項前段規定，諭知如
    易科罰金之折算標準；並就拘役刑部分定其應執行刑，及諭
    知如易科罰金之折算標準。至公訴人雖具體求處被告應執行
    有期徒刑1年，惟被告於本院審理時已表達悔意，且證人即
    被害人乙○○於本院審理時，經本院詢之對被告刑度有何意
    見，被害人乙○○則稱：請從輕量刑等語（見本院審判筆錄
    ），是本院認以宣告如主文所示之刑即屬適當，併此敘明。
五、公訴意旨另以被告於95年5 月9 日，以「我是混竹聯幫的，
    身上沒有錢可以花用，如果不給錢，就要讓你關店」等語，
    向證人乙○○強索現金30元，因認被告此部分，亦涉有刑法
    第346 條第1 項之恐嚇取財犯嫌。按犯罪事實應依證據認定
    之，無證據不得認定犯罪事實；又不能證明被告犯罪者，應
    諭知無罪之判決，刑事訴訟法第154 條第2 項、第301 條第
    1 項分別定有明文。本件公訴人認被告涉有此部分犯嫌，主
    要係以證人乙○○之證述為其論據。惟證人乙○○於本院審
    理時，經詳細思考確認被告恐嚇取財之次數及金額後，已詳
    為證述應如附表肆所示，是就公訴人起訴之此部分，已無實
    據可佐。惟公訴人認此部分與前揭起訴經本院認定恐嚇取財
    有罪部分，有連續犯之裁判上一罪關係，本院不另為無罪之
    諭知。
據上論斷，應依刑事訴訟法第299 條第1 項前段、第300 條，刑
法第2 條第1 項前段、修正刪除前刑法第56條、刑法第304 條第
1 項、第339 條第1 項、第346 條第1 項、修正前刑法第47條、
第41條第1 項前段、第51條第6 款，罰金罰鍰提高標準條例第 1
條前段，修正刪除前罰金罰鍰提高標準條例第2 條，判決如主文
。

├──┼─────┼─────────────────────────┤
│ １ │連續犯    │被告行為後，刑法第56條連續犯之規定，業經刪除，則被│
│    │          │告之犯行，因行為後新法業已刪除連續犯之規定，此刪除│
│    │          │雖非犯罪構成要件之變更，但已顯已影響行為人刑罰之法│
│    │          │律效果，自屬法律有變更，且比較新、舊法結果，以舊法│
│    │          │論以連續犯對被告較為有利，應依刑法第2條第1項前段，│
│    │          │適用舊法之規定                                    │
├──┼─────┼─────────────────────────┤
│ ２ │累  犯    │被告不論依修正前之刑法第47條，抑或修正後之刑法第47│
│    │          │條第 1項，均構成累犯，且均應加重其刑，對被告而言並│
│    │          │無有利或不利之情形，不生任何影響，應依刑法第2條第1│
│    │          │項前段，適用舊法之規定                            │
├──┼─────┼─────────────────────────┤
│ ３ │易科罰金  │刑法第41條第1項前段關於得易科罰金之規定，修正前係 │
│    │          │以「因身體、教育、職業、家庭之關係或其他正當事由，│
│    │          │執行顯有困難」為其要件，新法則刪除此要件，修正理由│
│    │          │說明上述規定限制過嚴，欠缺彈性，與易科罰金制度旨在│
│    │          │救濟短期自由刑流弊之旨趣不符，且刪除該要件後使檢察│
│    │          │官得視具體個案審酌宜否易科罰金，更符告易科罰金制度│
│    │          │之意旨。另修正前刑法第41條第1項前段配合罰金罰鍰提 │
│    │          │高標準條例第2條（現已刪除）之規定，係以「銀元300元│
│    │          │即新台幣900元折算1日」為易科罰金之折算標準，至修正│
│    │          │後之刑法第41條第1項乃係以「新台幣1,000元、2,000元 │
│    │          │或3,000元折算1日」為易科罰金之折算標準。而觀諸歷年│
│    │          │來之實務運作情形，在法院宣告易科罰金折算標準之案件│
│    │          │，檢察官執行時乃以准予易科罰金為原則，不准易科罰金│
│    │          │為例外，是以比較修法前後就易科罰金准否及折算標準之│
│    │          │規定，應認修正後刑法第41條第1項前段提高易科罰金之 │
│    │          │折算標準之適用，對於行為人而言較為不利，依刑法第2 │
│    │          │條第1項前段之規定，應適用舊法之規定。             │
├──┼─────┼─────────────────────────┤
│ ４ │主刑罰金刑│刑法第33條第5款有關罰金刑部分，在新法部分係規定為 │
│    │部分      │罰金：新臺幣1,000元以上，以百元計算之。而舊法時則 │
│    │          │係規定為罰金：1元以上。經比較修法前後之規定，應認 │
│    │          │修正前刑法第33條第5款之規定，對行為人有利。       │
├──┼─────┼─────────────────────────┤
│ ５ │定應行刑  │刑法第51條第6 款有關拘役刑定應執行部分，在舊法係規│
│    │          │定不得逾4 個月；而在新法部分則係規定為不得逾120 日│
│    │          │。經比較修法前後之規定，僅係文字調整，對被告而言並│
│    │          │無有利或不利之情形，不生任何影響，應依刑法第2 條第│
│    │          │1項 前段，適用舊法之規定                          │
