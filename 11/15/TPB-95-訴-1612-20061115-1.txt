  (二)被告主張之理由：
    1.按海關進口稅則號別第8529.90.90號為「其他專用或主要
      用於第8525至8528節所屬器具之零件」，第1 欄稅率為Fr
      ee；稅則號別第8525.30.00號為「電視攝影機」，第1 欄
      稅率為5%。本案貨物攝影照相模組（Camera Module ）由
      鏡頭、濾光片及CMOS感測元件等組件所構成，已具備「攝
      影機」之主要特徵，被告核列稅則第8525.30.00號項下，
      並無不合。
    2.關於訴狀理由略稱：原告報運進口系爭貨物Sensor Modul
      e （或稱Camera Module ）係用於數位個人助理PDA 之影
      像感測模組，並非電視攝影機。就各該關鍵零組件占數位
      相機之製造成本比重而言，Camera Module 約占50% ，DS
      P （單晶體影像處理器晶片）占30% 至40% ，CPU （中央
      處理器）與電阻電容等其他零組件占10% ～20% 。Camera
      Module僅有塑膠鏡座holder、鏡片組lens、紅外線濾膜IR
      Filter、影像感測晶片CMOS Sensor IC、被動元件、載版
      及軟質PC板，僅具感測影像之功能而不具有將影像轉換為
      數位影像訊號之拍攝功能。數位相機之主要特性在於拍攝
      、儲存及播放3 項主要特性，系爭Camera Module 須併同
      DSP 始能將所感測到的光影加以處理及轉換為數位影像訊
      號，以完成拍攝影像功能；經處理之數位影像訊號，須透
      過CPU 始能進行影像處理控制及儲存，顯見系爭貨物不具
      電視攝影機之完整功能。參酌以往財政部78年8 月26日台
      財稅字第780189797 號函等核釋及最高行政法院91年判字
      第2372號及改制前行政法院85年度判字第978 號等判決意
      旨，系爭貨物宜依其主要功能歸列稅則第8529.90.90號，
      稅率為0%。又美國海關將CCD Camera Module 歸列稅則第
      8529.90 目下，以及將Digital Image Capture Device歸
      列稅則第8525.40 目下，益證本件被告適用稅則號別，顯
      有誤解等語，資為爭議。
    3.答辯理由：
      (1)查系爭貨物原申報貨物名稱為Sensor Module （或稱Ca
        mera Module ），該Camera Module 主要由鏡頭（Lens
        ）、CMOS感測元件（CMOS Sensor ）、紅外線濾膜（IR
        filter）、被動元件（電感、電容）及軟質印刷電路板
        等零組件所組成，可裝置於PDA （數位個人助理）、Sm
        art Phone （無線手機）…等產品上，使該等產品具有
        拍照攝影之功能。依規格表所載其解析度範圍（Resolu
        tion ranges ）為1304×968 （即130 萬畫素），可以
        每秒15至120 格拍攝各種連續動態影像如VGA 、SXGA、
        QCIF等，並以8-bit 之YUV/Ycb Cr 4：2 ：2 、RGB 4
        ：2 ：2 及Raw RGB Data等Video 影像信號輸出至PDA
        或無線手機等產品內，以Motion JPEG 或MPEG4 等格式
        儲存於該等PDA 或無線手機產品內部之記憶體，其本身
        並無記憶體供儲存。據上，系爭Camera Module 已具備
        Camera之完整性，又查此類Camera Module 大部分均內
        建DSP 或ISP （影像處理單元，Image Signal Process
        or），此觀諸系爭貨物之訊號輸出係以YUV/Ycb Cr、RG
        B 或Raw RGB Data等Video 格式至明。縱來貨未具DSP
        或ISP 以處理所攝取之影像，亦可藉由PDA 或無線手機
        內部之CPU （中央處理單元）進行影像之處理。故系爭
        Camera Module 既含鏡頭、濾光膜及最主要之CMOS感測
        元件等組件，業已具備「攝影機」之主要特徵，參據國
        際商品統一分類制度註解（下稱HS註解）中文版第1271
        頁第22行有關稅則第8525.30 目「電視攝影機」之詮釋
        ：「本組包括電視攝影機，不論是否裝有遙控鏡頭及光
        圈之裝置，…」，縱使系爭貨物係供裝置於PDA 或無線
        手機上，亦應歸列其專屬稅則第8525.30.00號項下。
      (2)次查稅則第8525.30 目之「電視攝影機」，參據HS註解
        第1271頁第23行之詮釋為包括報導、工業、科學及管理
        交通用之各種視訊攝影機，係可拍攝連續影像視訊（Vi
        deo ）並予傳送至Monitor 或Display ，本身無儲存功
        能。本案貨物依規格表說明不具錄存影像之功能，自以
        歸列稅則第8525.30.00號下最為妥適。且類同貨物Came
        ra Module for Mobile Phone（行動手機用之攝影照相
        模組）業經財政部關稅總局稅則處核釋歸列稅則第8525
        .30.00號，有被告94年1 月10日（94）北關字003 號進
        口稅則分類疑問及解答在卷可稽。原告所舉財政部78年
        8 月26日台財稅字第780189797 號等2 函以及2 份行政
        法院判決書，係分別針對自動販賣機、工業用微波爐應
        否課徵貨物稅所作之核釋，以及對含浸紙及Flux等物品
        稅則分類爭議所作之判決，該等貨品與本案Camera Mod
        ule 毫不相干，要無比附援引之餘地。至原告所稱美國
        海關將CCD Camera Module 歸入稅則第8529.90 目，以
        及將Digital Image Capturing Device歸入稅則第8525
        .40 目乙節，經查前者CCD Camera Module 係以CCD （
        電荷藕合元件）感測，且係作為彩色閉路電視系統攝影
        機之用；而後者Digital Image Capturing Device（數
        位影像擷取元件）則可攝影像並儲存（Store ）影像。
        系爭來貨則係裝置於本身已具備CPU （中央處理單元）
        之PDA 或無線手機上，以CMOS（互補式金屬氧化物半導
        體）感測之Camera Module ，僅具拍攝影像並無儲存影
        像功能，核上開貨物無論構造、原理、用途均有不同，
        不宜比照援引，況美國海關之見解對本案並無任何拘束
        力。再查HS註解所稱之視訊攝影機僅須具備動態影像拍
        攝之功能即已足，有無具備儲存（記憶）及播放兩項功
        能則非所問。系爭Digital Camera Module 具備拍攝動
        態影像經處理後以各種視訊（YUV/Ycb Cr、RGB 、Raw
        RGB Data）輸出之功能，核屬「電視攝影機」範疇，被
        告核定稅則第8525.30.00號，並無不妥。原告所稱系爭
        Camera Module 未具「儲存」及「播放」兩項功能，不
        具電視攝影機之完整功能，應屬第8525至第8528節所屬
        器具之零件云云，顯不瞭解海關稅則分類規則，委無足
        採。
    理  由
一、按海關進口稅則號別第8529.90.90號為「其他專用或主要用
    於第8525至8528節所屬器具之零件」，第1 欄稅率為Free；
    稅則號別第8525.30.00號為「電視攝影機」，第1 欄稅率為
    5%。
二、本件原告於93年11月9 日至93年12月23日間報運進口SENSOR
    MODULE貨物15批，原申報稅則號別為第8529.90.90號，稅率
    為Free；案經被告事後依據原告補送型錄審核結果，以來貨
    應為攝影照相模組（CAMERA MODULE ），乃改歸列稅則第85
    25.30.00號項下，按稅率5%課徵關稅。原告不服，主張系爭
    貨物Sensor Module （或稱Camera Module ）係用於數位個
    人助理PDA 之影像感測模組，並非用於其他電視攝影機，且
    未連同構成數位相機主要特性之其他關鍵零組件一併進口。
    就各該關鍵零組件占數位相機之製造成本比重而言，Camera
    Module約占50% ，DSP （單晶體影像處理器晶片）占30% 至
    40% ，CPU （中央處理器）與電阻電容等其他零組件占10%
    至20% ，Camera Module 僅有塑膠鏡座holder、鏡片組lens
    、紅外線濾膜IR Filter 、影像感測晶片CMOS Sen sor IC
    、被動元件、載版及軟質PC板，僅具感測影像之功能而不具
    有將影像轉換為數位影像訊號之拍攝功能，除非系爭貨物係
    與構成數位相機主要特性之其他關鍵零組件如DPS 、甚至CP
    U 一併進口可以稅則第8525.30.00號，稅率為5%課徵較無疑
    義外，否則Camera Module 僅得適用稅則第8529.90.90，稅
    率0%。又美國海關將CCD Camera Module 歸列稅則第8529.9
    0 目下，與被告之認定不同。且數位相機3 項主要特性「拍
    攝」、「儲存」及「播放」中，系爭貨物Camera Mod ule須
    搭配DSP 始能達到「拍攝之功能」，遑論其他2 項主要特性
    「儲存」及「播放」，依據海關進口稅則解釋準則之準則二
    （甲）前段「稅則號別中所列之任何一種貨品，應包括該項
    貨品之不完整或未完成者在內，惟此類不完整或未完成之貨
    品，進口時需已具有完整或完成貨品之主要特性。該稅則號
    別亦應包括該完成之貨品（或由於本準則而被列為完整或完
    成者），而於進口時未組合或經拆散者」之規定，系案貨物
    不應適用攝影機之稅則號別課稅云云。是本件應審酌者厥為
    ：系爭貨物之稅則號別為原告原申報之第8529.90.90 號 ，
    稅率為Free？抑應改歸列為第8525.30.00號，稅率5%？
三、經查：
  (一)系爭貨物Camera Module 主要由鏡頭（Lens）、CMOS感測元
    件（CMOS Sensor）、 紅外線濾膜（IR filter）、 被動元
    件（電感、電容）及軟質印刷電路板等零組件所組成，可裝
    置於PDA （數位個人助理）、Smar t Phone（無線手機）等
    產品上，使該等產品具有拍照攝影之功能，依規格表所載，
    其解析度範圍（Resolution range s）為1304×968 （即13
    0 萬畫素），可以每秒15至120 格拍攝各種連續動態影像如
    VGA 、SXGA、QCIF等，並以8-bit 之YUV/Ycb Cr 4：2 ：2
    、RGB 4 ：2 ：2 及Raw RGB Data等Video 影像信號輸出至
    PDA 或無線手機等產品內，以Moti on JPEG或MPG4等格式儲
    存於該等PDA 或無線手機產品內部之記憶體，其本身並無記
    憶體供儲存，足見系爭貨物Camera Module 已具備Camera之
    完整性。
  (二)又此類Camera Module 大部分均內建DSP 或ISP （影像處理
    單元，Image Signal Pro cessor）， 此觀諸系爭貨物之訊
    號輸出係以YUV/Ycb Cr、RG B或Raw RGB Data等Video 格式
    至明，縱來貨未具DSP 或ISP 以處理所攝取之影像，亦可藉
    由PDA 或無線手機內部之CPU （中央處理單元）進行影像之
    處理。系爭貨物Camera Mod ule既含鏡頭、濾膜及最主要之
    CMOS感測元件等組件，業已具備「攝影機」之主要特徵，依
    據HS註解中文版第1271頁第22行有關稅則第8525.30 目「電
    視攝影機」之詮釋：「本組包括電視攝影機，不論是否裝有
    遙控鏡頭及光圈之裝置，…」故縱系爭貨物係供裝置於PDA
    或無線手機上，亦應歸列其專屬稅則第8525.30.00號項下。
  (三)次按稅則第8525.30 目之「電視攝影機」，參據HS註解第12
    71頁第23行之詮釋，為包括供報導、工業、科學及管理交通
    用之各種視訊攝影機，係可拍攝連續影像視訊（Video ）並
    予傳送至Monitor 或Disp lay，本身無儲存功能。亦即HS註
    解所稱之視訊攝影機，僅須具備動態影像拍攝之功能即已足
    ，有無具備儲存（記憶）及播放兩項功能則非所問，系爭Di
    gital Camera Module 具備拍攝動態影像經處理後以各種視
    訊（YUV/Ycb Cr、RGB 、Raw RGB Data）輸出之功能，核屬
    「電視攝影機」範疇，依其規格表說明不具錄存影像之功能
    ，自仍以歸列稅則第8525.30.00號項下最為妥適。
  (四)又按類同貨物Camera Module for Mobile Phone（行動手機
    用之攝影照相模組）業經財政部關稅總局稅則處（下稱稅則
    處）核釋歸列稅則第8525.30.00號，有被告94年1 月10日（
    94）北關字003 號「進口稅則分類疑問及解答」在卷可稽。
  (五)至原告稱美國海關將Camera Module 歸入稅則第8529.90 目
    乙節，經查該案之Camera Module 係以CCD （電荷藕合元件
    ）感測，且係作為彩色閉路電視系統攝影機之用，然系爭貨
    物則係裝置於本身已具備CPU （中央處理單元）之PDA 或無
    線手機上，以CMOS（互補式金屬氧化物半導體）感測之Came
    ra Module ，僅具拍攝影像並無儲存影像功能，核與上開貨
    物無論構造、原理、用途均有不同，不宜比照援引；況美國
    海關之見解對本案並無任何拘束力，故原告所稱要不足採。
  (六)原告另稱：依財政部78年8 月26日及80年6 月13日台財稅字
    第780189797 號函及第800706671 號函、最高行政法院91年
    判字第2372號判決，及改制前行政法院85年判字第978 號判
    決意旨，貨物應適用何種之稅則號別，仍須視其主要功能而
    定，而系爭貨物Sensor Module （或稱Camera Module ）係
    用於數位個人助理PDA 之影像感測模組，要難認具有電視攝
    影機之完整功能，亦非屬電視攝影機零件。又系爭貨物所佔
    數位相機成本之比例並不高，更不得視為已具有相機之完整
    性云云。惟查：
    1.本案貨品（按指Camera Module for Mobile Phone《行動
      手機用之攝影照相模組》）已具有攝取連續影像轉成類比
      信號輸出，雖未具錄影功能，惟已達『電視攝影機』之主
      要特徵，依據H.S.解釋準則二（甲）規定，應歸列稅則第
      8525.30. 00 號。」，有稅則處復被告94年1 月10日（94
      ）北關字003 號進口稅則分類疑問及解答附卷可稽。
    2.又此類Camera Module 大部分均內建DSP 或ISP （影像處
      理單元，Image Signal Processor），此觀諸系爭貨物之
      訊號輸出係以YUV/Ycb Cr、RGB 或Raw RGB Data等Video
      格式至明，縱來貨未具DSP 或ISP 以處理所攝取之影像，
      亦可藉由PDA 或無線手機內部之CPU （中央處理單元）進
      行影像之處理。系爭貨物Camera Module 既含鏡頭、濾膜
      及最主要之CMOS感測元件等組件，業已具備「攝影機」之
      主要特徵，參據上開HS註解中文版第1271頁第22行以下有
      關稅則第8525.30 目「電視攝影機」之詮釋及稅則處對類
      此貨物所為進口稅則分類疑問及解答，被告將系爭貨物改
      歸列稅則第8525.30.00號（電視攝影機），核無不合。
    3.至財政部78年8 月26日及80年6 月13日台財稅字第780189
      797 號函及第800706671 號函，係分別針對自動販賣機、
      工業用微波爐應否課徵貨物稅所作之核釋，最高行政法院
      91年判字第2372號判決，及改制前行政法院85年判字第97
      8 號判決，係分別對含浸紙及Flux等物品稅則分類爭議所
      作之判決，該等貨品核與本案Camera Module 毫不相干，
      要無比附援引之餘地。
  (七)原告又稱美國海關將CCD Camera Module 歸入稅則第8529.
    90目，以及將Digital Image Capturing Device歸入稅則第
    8525.40 目乙節，經查前者CCD Camera Module 係以CCD （
    電荷藕合元件）感測，且係作為彩色閉路電視系統攝影機之
    用；而後者Digital Image Capturing Device（數位影像擷
    取元件）則可攝影像並儲存（Store ）影像。系爭來貨則係
    裝置於本身已具備CPU （中央處理單元）之PDA 或無線手機
    上，以CMOS（互補式金屬氧化物半導體）感測之Camera Mod
    ule ，僅具拍攝影像並無儲存影像功能，核上開貨物無論構
    造、原理、用途均有不同，不宜比照援引，況美國海關之見
    解對本案並無拘束力。。
四、綜上，被告以系案貨物為攝影照相模組（CAMERA MODULE ）
    ，改歸列稅則號別第8525.30.00號（電視攝影機），按稅率
    5%課徵關稅，並無違誤。訴願決定予以維持，亦無不合。原
    告徒執前詞，訴請撤銷，為無理由，應予駁回。
五、兩造其餘攻擊防禦方法均與本件判決結果不生影響，故不逐
    一論述，併此敘明。
據上論結，本件原告之訴為無理由，爰依行政訴訟法第98條第3
項前段，判決如主文。
